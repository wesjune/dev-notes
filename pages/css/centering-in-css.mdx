# Centering in CSS

## Grid

```css
.content-center {
  display: grid;
  place-content: center;
}
```

## Flex

```css
.gentle-flex {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

```css
.autobot {
  display: flex;
}

.autobot > * {
  margin: auto;
}
```

## Position

```css
.parent {
  position: relative;
}

.child {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```
